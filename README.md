# NavMan
Keyboard navigation for macOS — Move your text cursor[carret] anywhere inside your computer without touching the arrow keys.
Hosted inside [Boostype](https://boostype.com)

Made by [Daniel Guevara](https://github.com/core-sixteen)

---

## Install

Copy and paste the following code inside your terminal:

```bash
curl -fsSL https://boostype.com/navman/navmanInstaller.sh | bash
```

## Usage

Run in Terminal:

```bash
navman
```

The app will ask for your credentials, use the following:

> **Testing credentials** (for demo purposes only):
> - Email: `test@navman.com`  
> - App key: `free_trial`

## How it works

Hold `Space` for more than 1 second to enter **navigation mode**, then:

| Key | Action |
|-----|--------|
| `I` | ↑ Up |
| `J` | ← Left |
| `K` | ↓ Down |
| `L` | → Right |
| `A` | Shift (select) |
| `S` | Option (move by word) |
| `D` | Command (move by line) |

Tap `Space` quickly → normal space character.

- Accessibility permissions (the installer will guide you)
